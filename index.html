Key Logic: Window Functions

WITH ActivityDelta AS (
    SELECT user_id, LAG(active_days) OVER 
    (PARTITION BY user_id ORDER BY month) as prev_month
    FROM user_logs
)

Implementation of LAG() avoided costly self-joins, optimizing performance on large transaction sets.
